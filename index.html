CompTIA Network+ (N10-009) Practice Questions

Complete Answer Key with Detailed Explanations

Objective 1.1: OSI Model

Question 1

Which of the following steps of the troubleshooting methodology would most likely include checking through each level of the OSI model after the problem has been identified?

A. Establish a theory.

B. Implement the solution.

C. Create a plan of action.

D. Verify functionality.

Show Answer & Explanation ▶

Correct Answer: A. Establish a theory.

Why this is correct: The "Establish a theory" step involves making an educated guess about the problem's cause. The OSI model is a powerful tool for this. A common troubleshooting method is to form theories layer by layer (e.g., "Is it a physical cable issue?" - Layer 1 theory; "Is it an IP address issue?" - Layer 3 theory). This structured approach helps narrow down possibilities efficiently.

Why others are incorrect:

  • Implement the solution: This happens after you've tested your theory and confirmed the cause.
  • Create a plan of action: This step comes just before implementation, where you decide *how* to fix the problem.
  • Verify functionality: This is the final step, performed after the fix is applied to ensure the problem is resolved.

Question 2

Which of the following layers of the OSI model is responsible for end-to-end encryption?

A. Presentation

B. Application

C. Session

D. Transport

Show Answer & Explanation ▶

Correct Answer: A. Presentation

Why this is correct: The Presentation Layer (Layer 6) is responsible for data translation, formatting, and syntax. This includes key security functions like encryption and decryption (e.g., SSL/TLS). Its job is to ensure that data sent from one system's application layer is readable by the destination system's application layer, and encryption is a critical part of that process.

Why others are incorrect:

  • Application (Layer 7): Provides the user interface to the network (e.g., browser) but relies on lower layers for encryption.
  • Session (Layer 5): Manages the dialogue between devices but doesn't handle the data's format or encryption.
  • Transport (Layer 4): Manages end-to-end data delivery (e.g., TCP/UDP) but not its encryption.

Question 3

Which of the following is a characteristic of the application layer?

A. It relies upon other layers for packet delivery.

B. It checks independently for packet loss.

C. It encrypts data in transit.

D. It performs address translation.

Show Answer & Explanation ▶

Correct Answer: A. It relies upon other layers for packet delivery.

Why this is correct: The Application Layer (Layer 7) is the highest layer in the OSI model. It's where user-facing applications like web browsers and email clients operate. This layer creates the data to be sent but has no knowledge of the underlying network. It completely depends on the lower layers (Transport, Network, Data Link, Physical) to handle the complex tasks of segmentation, addressing, routing, and physical transmission of the data.

Why others are incorrect:

  • Packet loss checking: This is a function of the Transport Layer (Layer 4), specifically by TCP.
  • Encryption: This is primarily a function of the Presentation Layer (Layer 6).
  • Address translation: Logical-to-physical address translation (IP to MAC) is done at the Network and Data Link layers (ARP), while Network Address Translation (NAT) is a Network Layer function.

Question 4

Which of the following best describes the transmission format that occurs at the transport layer over connectionless communication?

A. Datagram

B. Segment

C. Frames

D. Packets

Show Answer & Explanation ▶

Correct Answer: A. Datagram

Why this is correct: At the Transport Layer (Layer 4), the Protocol Data Unit (PDU) for the connectionless protocol, UDP (User Datagram Protocol), is called a datagram. It's a self-contained package of data with source and destination port information.

Why others are incorrect:

  • Segment: This is the PDU for TCP, the connection-oriented protocol at the Transport Layer.
  • Frames: This is the PDU for the Data Link Layer (Layer 2).
  • Packets: This is the PDU for the Network Layer (Layer 3).

Question 5

Which of the following OSI model layers can utilize a connectionless protocol for data transmission? (Choose two)

A. Physical

B. Network

C. Transport

D. Application

Show Answer & Explanation ▶

Correct Answer: B. Network and C. Transport

Why this is correct:

  • The Network Layer (Layer 3) uses the Internet Protocol (IP), which is inherently connectionless. It sends packets without first establishing a connection.
  • The Transport Layer (Layer 4) offers both options. It can provide connectionless service using UDP (User Datagram Protocol) or connection-oriented service using TCP (Transmission Control Protocol).

Why others are incorrect:

  • Physical (Layer 1): Deals with bits and signals, not protocols in this sense.
  • Application (Layer 7): While some application protocols use connectionless transport (like DNS using UDP), the layer itself isn't defined by this characteristic.

Question 6

Which of the following layers in the OSI model is responsible for establishing, maintaining, and terminating connections between nodes?

A. Physical

B. Network

C. Session

D. Transport

Show Answer & Explanation ▶

Correct Answer: C. Session

Why this is correct: This is the primary function of the Session Layer (Layer 5). It acts as the "dialogue controller" for the network. It creates a session, manages the back-and-forth communication between the two devices, and ensures the session is closed properly when the communication is complete.

Why others are incorrect:

  • Transport (Layer 4): While TCP establishes a "connection," the Session Layer manages the overall communication dialogue that uses that connection.
  • Network (Layer 3): Handles routing packets, not managing the state of a connection.
  • Physical (Layer 1): Deals only with the physical transmission of bits.

Question 7

A network administrator is troubleshooting an issue with a user's IP address. Which of the following layers of the OSI model should be considered?

A. Data link

B. Network

C. Session

D. Presentation

Show Answer & Explanation ▶

Correct Answer: B. Network

Why this is correct: The Network Layer (Layer 3) is where logical addressing and routing happen. The Internet Protocol (IP) address is the primary form of logical addressing used in modern networks. Therefore, any problem related to IP addresses, subnets, gateways, or routing is a Layer 3 issue.

Why others are incorrect:

  • Data Link (Layer 2): Deals with physical (MAC) addresses, not logical (IP) addresses.
  • Session (Layer 5): Manages communication sessions.
  • Presentation (Layer 6): Handles data formatting and encryption.

Question 8

Which of the following is the OSI model layer that uses a connectionless protocol for packet delivery?

A. Physical

B. Network

C. Transport

D. Application

Show Answer & Explanation ▶

Correct Answer: B. Network

Why this is correct: The Network Layer (Layer 3)'s primary protocol is the Internet Protocol (IP). IP is fundamentally connectionless; it provides a "best-effort" delivery service, meaning it sends packets (datagrams) toward a destination without establishing a prior connection or guaranteeing that they will arrive. Reliability is left to higher-layer protocols like TCP.

Why others are incorrect:

  • Transport (Layer 4): Can be connectionless (UDP) or connection-oriented (TCP), so it's not exclusively one or the other.
  • Physical (Layer 1): Is concerned with transmitting bits, not the connection state of protocols.
  • Application (Layer 7): Uses the services of lower layers; it doesn't define the connection state itself.

Objective 1.2: Networking Appliances, Applications, and Functions

Question 9

Which of the following technologies is the best choice to listen for requests and distribute user traffic across web servers?

A. Router

B. Switch

C. Firewall

D. Load balancer

Show Answer & Explanation ▶

Correct Answer: D. Load balancer

Why this is correct: A load balancer is specifically designed to distribute network traffic across multiple servers to ensure no single server becomes overwhelmed. It improves responsiveness and increases availability of applications.

Why others are incorrect:

  • Router: Connects different networks and routes traffic between them, but doesn't distribute traffic across servers.
  • Switch: Connects devices within the same network and forwards traffic based on MAC addresses.
  • Firewall: Provides security by filtering traffic but doesn't distribute load.

Question 10

Which of the following would allow a network administrator to analyze attacks coming from the internet without affecting latency?

A. IPS

B. IDS

C. Load balancer

D. Firewall

Show Answer & Explanation ▶

Correct Answer: B. IDS

Why this is correct: An Intrusion Detection System (IDS) monitors network traffic for suspicious activity and generates alerts but doesn't take action to block traffic. Because it operates in passive mode (typically using port mirroring), it doesn't introduce latency like an active protection system would.

Why others are incorrect:

  • IPS (Intrusion Prevention System): Actively blocks threats but adds latency as it sits inline with traffic.
  • Load balancer: Distributes traffic but doesn't analyze attacks.
  • Firewall: Actively filters traffic, which can add latency.

Question 20

Which of the following technologies is most appropriate for a business that requires high-speed access to frequently used web content, such as images and videos?

A. CDN

B. SAN

C. Firewall

D. Switch

Show Answer & Explanation ▶

Correct Answer: A. CDN

Why this is correct: A Content Delivery Network (CDN) stores cached copies of web content (like images and videos) at geographically distributed edge servers. This places content closer to users, reducing latency and improving access speed for frequently requested materials.

Why others are incorrect:

  • SAN (Storage Area Network): Provides block-level storage access but doesn't optimize content delivery to end users.
  • Firewall: Provides security but doesn't accelerate content delivery.
  • Switch: Connects devices but doesn't cache or optimize content.
